# SPIRAL-iCode #

SPIRAL iCode compiler

### Dependencies ###
* OCaml 4.04+
* ocamllex
* menhir
* Core
* Coq-8.6

To install all required dependenceis:

    opam install menhir core stdint ppx_jane sexplib ppx_sexp_conv ppx_compare ppx_deriving sexp_pretty getopt coq coq-bignums
