# SPIRAL-iCode #

SPIRAL iCode compiler

### Dependencies ###
* OCaml 4.06.0
* ocamllex
* menhir
* Jane Street Core
* Jbuilder
* Coq-8.7

To install all required dependenceis:

    opam install jbuilder menhir core stdint ppx_jane sexplib ppx_sexp_conv ppx_compare ppx_deriving ppx_enumerate sexp_pretty getopt coq
