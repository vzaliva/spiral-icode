# SPIRAL-iCode #

SPIRAL iCode compiler

### Dependencies ###
* OCaml 4.04+
* OMake
* ocamllex
* menhir
* Core

To install all required dependenceis:

    opam install menhir core omake uint ppx_jane sexplib ppx_sexp_conv ppx_compare sexp_pretty getopt
